# AI Based Fake News Detection Tool

## Overview
This project detects whether a news article is Real or Fake using Machine Learning and Natural Language Processing (NLP).

## Features
- Add news articles
- View news dataset
- Train ML model
- Detect Fake/Real news
- Confidence score prediction
- Save trained model

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Joblib

## Machine Learning
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Files
- main.py
- news.csv
- model.pkl
- vectorizer.pkl

## Run

```bash
pip install pandas scikit-learn joblib
python main.py
```
