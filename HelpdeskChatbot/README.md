# AI Helpdesk Chatbot

## Overview
An AI-powered chatbot that answers employee questions using Machine Learning and intent classification.

## Features
- Add FAQs
- View FAQ dataset
- Train chatbot
- Ask questions
- Intent-based replies

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Joblib

## Machine Learning
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Files
- main.py
- faq.csv
- chatbot_model.pkl
- vectorizer.pkl

## Run

```bash
pip install pandas scikit-learn joblib
python main.py
```
