# Smart Recommendation System

## Overview
A content-based recommendation system that recommends movies based on user preferences using Machine Learning.

## Features
- Add movies
- View dataset
- Personalized recommendations
- Content-based filtering
- Similarity scoring

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Machine Learning
- TF-IDF Vectorizer
- Cosine Similarity

## Files
- main.py
- movies.csv

## Run

```bash
pip install pandas scikit-learn
python main.py
```
