# Student Performance Prediction

## Overview
Predicts student final performance using attendance, marks, and study hours.

## Features
- Add student records
- View records
- Delete records
- Predict performance
- Evaluate model
- Display graphs

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning
- Linear Regression

## Files
- main.py
- student_data.csv

## Run

```bash
pip install pandas matplotlib scikit-learn
python main.py
```
