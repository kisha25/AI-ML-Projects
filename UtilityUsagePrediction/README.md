# Utility Usage Prediction

## Overview
Predicts future utility usage using historical usage data with Machine Learning.

## Features
- Add utility records
- View records
- Delete records
- Predict future usage
- Evaluate model
- Display charts

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning
- Linear Regression

## Files
- main.py
- utility_data.csv

## Run

```bash
pip install pandas matplotlib scikit-learn
python main.py
```
